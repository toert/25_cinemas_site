Список фильмов API

API сайта

Руководство написано на примере языка Python 3

Данный API предоставляет вам 2 метода: получить список всех фильмов, представленных на сайте, и получить по каждому фильму информацию по отдельности

Список всех фильмов

При обращении к https://movies-digest.herokuapp.com/api/movies вы можете получить список всех фильмов, которые находятся в базе данных сайта. Этот список обновляется раз в 12 часов. Ответ приходит в виде json:

{"movies": [{"url": "http://www.afisha.ru/movie/226786/", "id": "226786"}, {..}, {..}]

Где "url" — ссылка на фильм на сайте Афиши, "id" — используется для получения информации о фильме

Информация по каждому фильму

При обращении к https://movies-digest.herokuapp.com/api/movies/movie_id вы можете получить информацию о фильме, который находится в базе данных сайта. Эта информация обновляется раз в 12 часов. Ответ приходит в виде json.

'name' — Имя фильма на кирилице
'rating' — рейтинг фильма на Афише
'rating_count' — количество проголосовавших
'review' — полный обзор на фильм от Афиши
'description' — краткое описание картины
'alt_name' — имя фильма на языке-оригинале
'genre' — жанр
'content' — оценка фильма по возрастному допуску
'url' — ссылка на Афишу
'id' — идентификатор
'image_url' — ссылка на изображение-обложку
'img_min' — ссылка на изображение-обложку разрешения 352х200


Пример обращения: 

        
        >>movie = requests.get(https://movies-digest.herokuapp.com/api/movies/222117).json()
        >>print(movie['name'])
        Ла-ла-ленд
